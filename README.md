## Usage

> This iOS mobile application brings together owners and pet sitters in a craigslist style format.

## Requirements

- React-Native 0.11.x
- Firebase 2.3.x

### Dependencies and Running

```
npm install
npm start
```
You must download xCode to run the program, and import the ios folder.
